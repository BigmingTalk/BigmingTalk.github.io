C语言常见的存储器相关错误总结

计算机系统
  • C语言
  • 指针
  • 错误

对于C语言，一个非常困难且容易出错的地方是管理和使用虚拟存储器。错误经常是在于错误源距离一段时间和空间的地方表现出来，使得查错也是一个相当头疼的事情。本文总结了程序员常常会犯的几种错误，以供参考。

间接引用坏指针

进程的虚拟地址空间并不是连续的，如果我们试图引用一个指向该进程不占用的虚拟地址，操作系统会以段异常中止我们的程序。
